# 📱 Как собрать APK одной кнопкой

## Способ 1: Через GitHub (Самый простой!)

### Шаг 1: Создайте репозиторий на GitHub

1. Зайдите на https://github.com
2. Нажмите зелёную кнопку **"New"**
3. Назовите репозиторий (например: `web3gram-messenger`)
4. Нажмите **"Create repository"**

### Шаг 2: Загрузите код

**Вариант А: Через GitHub Desktop (рекомендую для новичков)**

1. Скачайте GitHub Desktop: https://desktop.github.com
2. Установите и войдите в аккаунт
3. Нажмите **"Clone a repository"**
4. Выберите свой репозиторий и клонируйте
5. Скопируйте ВСЕ файлы проекта в папку репозитория
6. В GitHub Desktop напишите комментарий и нажмите **"Commit to main"**
7. Нажмите **"Push origin"**

**Вариант Б: Через веб-интерфейс GitHub**

1. Откройте свой репозиторий
2. Нажмите **"Add file" → "Upload files"**
3. Перетащите все файлы проекта
4. Нажмите **"Commit changes"**

### Шаг 3: Соберите APK

1. Откройте свой репозиторий на GitHub
2. Нажмите на вкладку **"Actions"** (иконка молнии ⚡)
3. Вы увидите workflow **"🔨 Build Android APK"**
4. Нажмите на него
5. Нажмите кнопку **"Run workflow"** (зелёная кнопка справа)
6. Подождите 5-10 минут ⏳

### Шаг 4: Скачайте APK

1. Когда сборка закончится (зелёная галочка ✓)
2. Нажмите на сборку
3. Прокрутите вниз до раздела **"Artifacts"**
4. Нажмите на **"Web3Gram-APK"** чтобы скачать
5. Распакуйте zip файл
6. Получите файл `app-debug.apk` - это ваше приложение!

---

## Способ 2: Автоматическая сборка

После первого пуша, GitHub Actions будет автоматически собирать APK при каждом изменении кода.

Просто:
1. Отредактируйте код (или попросите меня)
2. Запушьте изменения
3. Подождите 5-10 минут
4. Скачайте новый APK из Actions

---

## 📲 Установка APK на телефон

1. Скачайте `app-debug.apk` на телефон
2. Откройте файл
3. Если Android спросит - разрешите установку из неизвестных источников
4. Установите приложение
5. Готово! 🎉

---

## ⚠️ Важно!

При первом запуске Android может показать предупреждение:
> "Это приложение от неизвестного разработчика"

Это нормально для debug-версий. Нажмите **"Всё равно установить"**

Для публикации в Google Play нужна подпись ключом (release версия).

---

## 🆘 Если что-то пошло не так

1. Проверьте что все файлы загружены в GitHub
2. Откройте Actions и посмотрите логи ошибок (красные строки)
3. Если ошибка "npm install" - попробуйте запушить ещё раз
4. Если нужна помощь - напишите issue в репозитории
